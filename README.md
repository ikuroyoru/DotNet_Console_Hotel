- Projeto em desenvolvimento criado com o objetivo de construir, de forma incremental, um sistema de gerenciamento de hotel em C# (Console).

- A proposta é evoluir o projeto conforme avanço nos estudos em .NET, aplicando boas práticas de arquitetura, separação de responsabilidades e modelagem de domínio.

## O sistema terá como foco:

- Cadastro e autenticação de usuários
- Cadastro e listagem de hotéis
- Gerenciamento de quartos
- Criação e controle de reservas
- Sistema de avaliações vinculado a reservas concluídas
- Aplicação de regras de negócio (disponibilidade por período, validações, etc.)

- O projeto começou com arquitetura em camadas (Menu, Service, Repository e Domain) e será expandido futuramente com novas funcionalidades, possíveis testes automatizados e persistência de dados.
